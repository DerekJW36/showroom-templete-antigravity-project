# Recommended Antigravity Skills for Showroom Project

To elevate this high-ticket e-commerce showroom template to a production-ready, highly converting asset, I recommend running the project through the following Antigravity skills, in this specific order:

## 1. High-End Polish & UX Engineering
*   **`antigravity-design-expert`**
    *   *Why:* This skill specializes in spatial UI, glassmorphism, and GSAP animations. Since we are using Framer Motion and dark luxury aesthetics, this agent can refine the micro-interactions, add 3D depth to the product gallery, and ensure the "weightless" feel is perfect across all devices.
*   **`scroll-experience`**
    *   *Why:* High-ticket items sell through storytelling. This skill focuses on scroll-driven narrative devices, parallax effects, and smooth transitions. It will make the journey down the page feel like a cinematic cinematic experience rather than a standard scroll.

## 2. Conversion & Performance
*   **`page-cro` (Page Conversion Rate Optimization)**
    *   *Why:* Before driving traffic, this skill will diagnose the page's readiness to convert. It evaluates value proposition clarity, trust signals (crucial for high-ticket sales), and friction points to ensure users click "Reserve Artifact."
*   **`seo-page` & `seo-fundamentals`**
    *   *Why:* To ensure the template ranks well for luxury keywords and has perfect semantic structure, meta tags, and schema.org data (to show rich product snippets in Google).
*   **`web-performance-optimization`**
    *   *Why:* Heavy animations and high-resolution images can hurt load times. This skill will audit the Vite build, optimize asset loading, and ensure core web vitals are met so the luxury experience never stutters.

## 3. Operations & Deployment
*   **`e2e-testing`**
    *   *Why:* Adds Playwright automated tests to ensure the gallery, modal, and reservation flows never break during future updates.
*   **`vercel-deployment` or `appdeploy`**
    *   *Why:* When you're ready to sell or host this template, these skills will handle the production build and deployment to a permanent, lightning-fast edge network.
